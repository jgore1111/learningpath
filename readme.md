# LearningPath

## Crear cuenta en GitHub

- Nombrar repo

- Crear repo con ssh

- Creación de una llave (ohmyzsh: ssh keygen )

- Obtener la llave (cat ./.ssh/id_rsa.pub )[Siempre ponder .pub para solo mostrar llave pública]

- Copiar la llave a Git (en ícono, settings, ssh and gpg keys, new sshkey y copiar la llave)
- Clonar un repositorio en el Term (copiar el URL en T con `git clone + url`)
